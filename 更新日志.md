# 更新日志 (ChangeLog)
## [1.8.1] - 2026-05-20

### PWA 缓存治理与版本号提取自动化
- **后端拓宽劫持清理**：后端服务拓宽对 Service Worker 路由的劫持范围，增加对 `/service-worker.js` 的拦截，与历史的 `/sw.js`、`/registerSW.js` 统一实施 no-store 强力注销与客户端缓存清空，从服务端彻底抹除残留的旧 SW 缓存。
- **客户端版本绑定清理**：优化前端 `pwaCleanup.js` 清理逻辑，将本地 PWA 清理标识 Key 与应用版本号强绑定。当应用版本升级时，客户端自动执行且仅执行一次彻底的缓存清除并自动刷新，让新部署的内容对老用户无感生效。
- **自动提取更新日志版本**：优化前端 Vite 编译打包流程，在编译时通过正则表达式自动从《更新日志.md》中提取最新的发布版本号并注入全局常量，省去手动修改多处版本号的繁琐，实现版本同步的完全自动化。

### Android APK 正式打包链路
- 完成 Android APK 首包发布准备，版本统一升级为 `1.8.0`，`versionCode` 升级为 `180`。
